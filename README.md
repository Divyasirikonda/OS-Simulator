# OS Simulator in C++

This project is a collection of Operating System algorithms implemented using C++.

## Implemented Algorithms

### Page Replacement Algorithms
- FIFO
- LRU
- Optimal Page Replacement

### CPU Scheduling Algorithms
- FCFS
- SJF
- Round Robin

### Disk Scheduling Algorithms
- FCFS 
- SSTF
- SCAN
- C-SCAN

## Technologies Used
- C++
- VS Code
- Git & GitHub
